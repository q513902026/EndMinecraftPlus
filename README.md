# EndMinecraftPlus
基于<a href="https://github.com/iuli-moe/EndMinecraft">EndMinecraft</a>优化修改的Minecraft压测工具<br>
本工具只可用于服务器压力测试,请勿用于其他非法用途!否则造成的后果由使用者自行承担!

#### 支持方式: ####
1.单IP MOTD<br>
2.集群假人<br>
3.集群假人Tab<br>
4.Forge协议<br>
5.AntiCheat自定义MD5<br>

#### 修改内容: ####
1.重构大量核心代码<br>
2.大幅度性能优化<br>
3.修复Forge协议无法进服<br>
4.修复AntiCheat特定情况无法进服<br>
5.增加AntiCheat自定义MD5<br>
6.移除部分过时功能<br>
